# collection-seo

Generate SEO for shopify collections

To install dependencies:

```bash
bun install
```

To run:

```bash
bun run dev
```

## Usage

1. Setup env

Copy `.env` to `.env.local` and set tokens

2. Config stores:

Copy `stores.yaml.examle` to `stores.yaml`

Enter store names and tokens

3. `bun run start`
